# lexa-doctor

Run:

```bash
npx -y https://github.com/GoBeromsu/lexa/releases/download/lexa-v0.1.2/goberomsu-lexa-0.1.2.tgz doctor --vault <vault>
```

The command is advisory in v0 and exits 0 even when warnings are found.
