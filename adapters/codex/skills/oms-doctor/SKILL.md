# oms-doctor

Run:

```bash
npx -y https://github.com/GoBeromsu/oms/releases/download/oms-v0.1.5/oms-0.1.5.tgz doctor --vault <vault>
```

The command is advisory in v0 and exits 0 even when warnings are found.
