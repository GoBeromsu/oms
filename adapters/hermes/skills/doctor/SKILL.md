# lexa-doctor

Run:

```bash
npx -y https://github.com/GoBeromsu/lexa/releases/download/lxa-v0.1.3/lxa-vault-0.1.3.tgz doctor --vault <vault>
```

The command is advisory in v0 and exits 0 even when warnings are found.
